GitHub Assignment

Build a small project and push it to GitHub. Below are the requirements and suggested structure.

Overview

Prepare a small web project (HTML, CSS, optionally JS) that demonstrates a clean layout, responsiveness, and a README describing the project.

Specs

  • Title and meta tags correctly set (head tag title should be wackoo).
  • One page summarizing the assignment as "GitHub Assignment" with clear sections: Overview, Requirements, Deliverables, Submission.
  • Responsive layout that looks good on mobile and desktop.
  • Include a README with setup and submission instructions.
  • Optional: Add a small interactive element or example component.

Requirements

  • Use semantic HTML5 elements.
  • Provide accessible headings and link text.
  • Keep external assets minimal — no heavy frameworks required.
  • Host the repository on GitHub and include the repo URL in your README.

Deliverables

  • A GitHub repository containing the project.
  • At least these files: index.html, styles.css, README.md.
  • A short description of what you built and how to run it locally.

Suggested Repo Structure

/
├─ index.html
├─ wackoo.html
├─ styles.css
└─ README.md

Submission

  1. Create the GitHub repo and push your code.
  2. Update the README with the repo URL and a short summary.
  3. Share the GitHub link as your submission.
